The professional interactive checklist framework for **GeoFS-V3.9**. Designed to handle everything from generic general aviation procedures to high-fidelity airliner flows, this system ensures every phase of flight is executed with technical precision.

## ✨ Features

- **🎯 Pro Dynamic Detection**: Automatically detects the current aircraft airframe and loads the corresponding checklist profile (737, A320, etc.).
- **🧩 HUD Integration**: Seamlessly plugs into the **[CHECKS]** tab of the HUD Information Display for in-cockpit guidance.
- **📋 Interactive Logic**: Real-time "check-off" functionality with visual state feedback and session persistence.
- **🛡️ SafeInit Architecture**: Built on the V3.9 Core Library for rock-solid stability and zero-crash module initialization.
- **🚀 CDN Loader Architecture**: Instant updates via a lightweight loader pattern, ensuring you always have the latest checklist features.

## 🛠️ Installation

### Standalone (Tampermonkey)

1. Install the [Tampermonkey](https://www.tampermonkey.net/) extension.
2. Install the script via [**Greasy Fork**](https://greasyfork.org/en/scripts/581516-geofs-v3-9-checklist-system).
3. Refresh GeoFS and look for the **CHECKLIST** options in the HUD interface.
